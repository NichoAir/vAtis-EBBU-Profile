# BELUX vACC vATIS Profiles

This project contain the inofficial vATIS Profiles of the Belux vACC.  
To use this profiles you need the [vATIS](https://vatis.app/) programm.  
Once you imported the profiles to vATIS, the profiles are automatically updated, whenever you launch vATIS.  

## Getting Started as Controller
1. Download vATIS from https://vatis.app/  
2. Download the [Profile](https://github.com/NichoAir/vAtis-EBBU-Profile/releases). 
3. Import Profile to vATIS  
4. Select RWY config for the Airport and connect
Documentation: https://vatis.app/docs/  

Note: The Profiles are automatically updated, whenever you launch vATIS and an update is available.
